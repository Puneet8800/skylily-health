# skylily-health

![CI](https://github.com/Puneet8800/skylily-health/actions/workflows/ci.yml/badge.svg)
![License](https://img.shields.io/github/license/Puneet8800/skylily-health)

Service health monitoring CLI for Clawdbot and system services.

## Installation

```bash
npm install -g skylily-health
# or clone and link
git clone https://github.com/Puneet8800/skylily-health.git
cd skylily-health && npm install && npm link
```

## Usage

```bash
sky-health
```
